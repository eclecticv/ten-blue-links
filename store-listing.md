# Chrome Web Store listing copy

## Name (45 char max)
Ten Blue Links

## Short description (132 char max)
Removes AI Overviews, ads, and clutter from Google Search. No settings, no toggles — just the ten blue links you came for.

## Detailed description

Google Search used to be ten blue links. Now it's an AI Overview, three ads, a People Also Ask box, a knowledge panel, a "Things to know" widget, a Discussions section, an AI Mode tab, and — somewhere down there — the result you were actually looking for.

Ten Blue Links quietly removes everything between you and the results.

**What it hides:**
- AI Overviews and AI Mode entry points
- Sponsored results and shopping ads
- People Also Ask
- Discussions and Forums blocks
- Knowledge panel / right sidebar
- Featured snippet cards
- "Things to know" widgets

**What it keeps:**
- The ten blue links
- Search filter tabs (All, Images, Videos, Maps, News)
- Pagination and related searches at the bottom

**What it doesn't do:**
- No settings page
- No toolbar popup
- No telemetry
- No account, no login, no upsell
- No network requests beyond the page you're already loading

It's a CSS file and a manifest. That's the whole extension. Install it, forget it, search like it's 2010.

**Supporting publishers**

When you click through to a real article instead of reading an AI Overview, the publisher gets the page view, the ad impression, and the chance to earn a subscriber. Ten Blue Links exists because the open web is worth a click.

## Category
Productivity

## Permissions justification
This extension requests no permissions beyond `host_permissions` for `www.google.com`. It cannot read or modify any other site. There is no background script, no storage, no network access.

## Single-purpose declaration
Ten Blue Links has one purpose: hide non-result content from Google Search pages so users can see the organic results without visual interference.
